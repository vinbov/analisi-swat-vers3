Electron Container for Analisi S.W.A.T.

Note: If you are seeing this page, Electron is likely configured to load this local index.html file directly (e.g., via mainWindow.loadFile('index.html') in main.js).

To display your "Analisi S.W.A.T." Next.js application within this Electron window, ensure that:

  1. Your Next.js application is running (e.g., on http://localhost:9002).
  2. The main.js file in Electron is configured to load that URL, for example:
    mainWindow.loadURL('http://localhost:9002');

When loadURL is used with your Next.js app's address, this current HTML page (index.html) will not be the main content visible in the Electron window. The window will instead show your live Next.js application.

We are using Node.js -, Chromium -, and Electron -.
